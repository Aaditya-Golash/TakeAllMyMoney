# TakeAllMyMoney

Throw money. Get nothing. Climb a leaderboard.  
Retro vibe + Stripe Checkout. **Not gambling** (no prizes, no odds).

## Stack
Next.js (App Router) • TypeScript • Tailwind • Postgres (Supabase/Neon) • Stripe • Vercel

## Quickstart
```bash
# Node 20+, pnpm preferred
pnpm i
pnpm dev
